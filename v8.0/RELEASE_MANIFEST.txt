================================================================================
INSTALLATION & USAGE
================================================================================

Windows:
1. Extract the package
2. Run ym2163_piano_gui_v8.exe
3. Connect YM2163 hardware via USB
4. Load MIDI files and play

Linux/MinGW:
1. Extract the package
2. Run: bash build_v8.sh
3. Run the compiled executable
4. Connect YM2163 hardware via USB

================================================================================
SYSTEM REQUIREMENTS
